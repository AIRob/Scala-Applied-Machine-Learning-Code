#Raspberry Pi: Making Amazing Projects Right from Scratch!

This is the code repository for [Raspberry Pi: Making Amazing Projects Right from Scratch!](https://www.packtpub.com/application-development/raspberry-pi-making-amazing-projects-right-scratch?utm_source=github&utm_medium=repository&utm_campaign=9781787128491), published by Packt. It contains all the supporting project files necessary to work through the book from start to finish.

## Instructions and Navigation

All of the code is organized into folders. Each folder starts with a module number. For example, `Module 1`.
